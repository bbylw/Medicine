## 项目简介

循证医学（Evidence-Based Medicine, EBM）由加拿大临床流行病学家 David Sackett 教授于 1992 年正式提出，强调在医疗决策中结合**最佳研究证据**、**临床专业知识**与**患者价值观**三大支柱。

本项目以科普教育为目的，通过精心设计的视觉界面和清晰的信息架构，将循证医学的核心概念、证据等级体系、就医实践指南和 AI 赋能应用呈现给公众。

### 证据等级金字塔

Hero 区域展示五级证据等级体系：

```
        ┌─────────────┐
        │  Level 1    │  系统评价
        │  Level 2    │  随机对照试验
        │  Level 3    │  队列研究 / 病例对照
        │  Level 4    │  病例系列 / 病例报告
        │  Level 5    │  专家意见 / 机制推理
        └─────────────┘
          高 ← 证据质量 → 低
```

### 循证五步法（5A）

| 步骤 | 英文 | 说明 | 关键工具/概念 |
|------|------|------|--------------|
| 1 | **Ask** | 将临床问题结构化 | PICO 框架（Patient, Intervention, Comparison, Outcome） |
| 2 | **Acquire** | 系统检索最佳证据 | Cochrane、PubMed、MeSH 主题词 |
| 3 | **Appraise** | 严格评价证据质量 | GRADE 系统、偏倚风险评估 |
| 4 | **Apply** | 结合患者情况应用 | NNT/NNH、共同决策 |
| 5 | **Assess** | 评估实施效果 | 临床审计、结局追踪 |

### GRADE 证据分级

| 等级 | 符号 | 含义 |
|------|------|------|
| 高质量 | ⊕⊕⊕⊕ | 非常有信心效应估计值接近真实值 |
| 中等质量 | ⊕⊕⊕◯ | 可能接近，但也可能有显著差异 |
| 低质量 | ⊕⊕◯◯ | 真实值可能与估计值存在显著差异 |
| 极低质量 | ⊕◯◯◯ | 几乎无信心，估计值很不确定 |

### 常见医疗误区

站内收录 6 组循证辟谣，每组均标注证据来源：

1. **输液比吃药好得快** → WHO：能口服不肌注，能肌注不输液
2. **抗生素能治感冒** → Cochrane 系统评价：抗生素对病毒无效
3. **体检越多越好** → USPSTF：过度筛查导致假阳性和过度治疗
4. **贵的药比便宜的好** → RCT 证实仿制药与原研药生物等效
5. **维生素C预防感冒** → Cochrane（29项 RCT）：对普通人群无效
6. **权威说的就是对的** → 专家意见在证据等级中处于最低层级

同时解析 4 类影响决策的认知偏差：确认偏差、可得性偏差、锚定效应、安慰剂效应。

### 可靠信息来源

就医实践板块推荐的国际权威资源，分为两大类：

**系统评价与文献数据库：**

- **Cochrane Library** — 考克兰图书馆，系统评价金标准
- **PubMed / MEDLINE** — 美国国立医学图书馆文献数据库
- **UpToDate** — 临床决策支持系统

**国际权威临床指南：**

- **WHO** — 世界卫生组织指南
- **NICE** — 英国国家卫生与临床优化研究所指南
- **NCCN** — 美国国家综合癌症网络（肿瘤学）
- **ADA** — 美国糖尿病协会指南（内分泌/糖尿病）
- **AHA/ACC** — 美国心脏协会/美国心脏病学会指南（心血管）
- **ESC** — 欧洲心脏病学会指南（心血管）
- **USPSTF** — 美国预防服务工作组（筛查/预防）
- **IDSA** — 美国感染病学会指南（感染病学）

## 设计系统

### 美学方向

采用 **Industrial Sci-Fi × Luxury** 暗色风格——深色暖黑底色搭配精致的玻璃拟态卡片，营造高端学术与科技感并存的视觉体验。

### 配色方案

三色体系，遵循 60-30-10 法则：

| 角色 | 色名 | 主色值 | 用途 |
|------|------|--------|------|
| 🟣 主色 | **靛紫 (pri)** | `#6347c9` | 品牌标识、CTA 按钮、主要强调、证据相关 |
| 🟢 辅色 | **柔青 (teal)** | `#28b0a6` | 临床经验、辅助信息、对勾/确认 |
| 🟡 点缀色 | **琥珀金 (gold)** | `#f4c430` | 患者价值观、警告提示、渐变暖色 |
| ⬛ 底色 | **暖黑 (base)** | `#0f0d0b` | 页面背景 |
| 🔘 文本 | **暖灰 (warm)** | `#b8ae9e` | 正文文字 |

### 字体系统

| 用途 | 字体 | 场景 |
|------|------|------|
| Display 标题 | **Space Grotesk** | 大标题、按钮、数字 |
| 正文 | **Noto Sans SC** | 中文内容段落 |
| 衬线标题 | **Noto Serif SC** | 备用中文标题 |
| 等宽标签 | **JetBrains Mono** | TAG 标记、分类标签 |

### 视觉效果

- **玻璃拟态卡片** — `backdrop-filter: blur(24px)` + 半透明边框
- **网格背景** — 60px 间距微光网格，呼吸动画
- **噪点纹理** — SVG fractalNoise 全局覆盖，增加质感
- **辉光分割线** — 紫→金渐变 1px 分割
- **浮动光球** — 三色模糊圆形，鼠标视差跟踪（桌面端）
- **Mesh 渐变** — 多层径向渐变叠加背景氛围
